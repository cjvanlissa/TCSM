Chapter 17 Week 3 - Formative test

A formative test helps you assess your progress in the course, and helps you address any blind spots in your understanding of the material. If you get a question wrong, you will receive a hint on how to improve your understanding of the material.

Complete the formative test ideally after you’ve seen the lecture, but before the lecture meeting in which we can discuss any topics that need more attention.

Question 1: The minimum number of items for a CFA is…

Question 2: PCA and EFA both assume that indicator variables do not have measurement error

Question 3: Generally, more factor loadings are estimated in a EFA model than in a CFA model

Question 4: A researcher has developed a new questionnaire that should measure depression, and wants to know how many factors there are. Which technique would you use?

Question 5: Which of these methods is valid for choosing the correct number of factors in CFA?

Question 6: Rotation can be applied to the loadings of both EFA and CFA.

Question 7: In CFA, some parameters must be constrained in order to make the model identified.

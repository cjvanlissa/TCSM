4.4.13 Question 3.m

Perform this analysis as an ANCOVA.

Hint: Add -1 to a formula to drop the intercept.

Click for explanation To drop the intercept from the analysis, and estimate the marginal means for all viewcat categories, we can add -1 (minus the intercept) to the formula:

results_ancov <- aov(POSTNUMB~AGE+VIEWCAT-1, data)
